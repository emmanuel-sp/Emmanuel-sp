## Featured Projects

### [NagAI](https://github.com/emmanuel-sp/NagAI)
**AI accountability and goal execution platform**

A multi-service application built with **Next.js**, **Spring Boot**, **PostgreSQL**, and a **Python AI service**. Uses **gRPC** for synchronous service communication and **Redis Streams** for asynchronous workflows.

**Highlights**
- Full-stack architecture across frontend, backend, and AI services
- gRPC-based communication between services
- Async workflow orchestration with Redis Streams
- Authentication, persistence, and production-minded service boundaries

---

### [Distributed Hash Table](https://github.com/emmanuel-sp/Distributed-Hash-Table)
**Chord-inspired distributed hash table in C++**

Implemented a ring-based DHT over **TCP sockets** with dynamic node joins, routed lookups, and key migration across the ring.

**Highlights**
- Distributed routing and lookup logic
- Dynamic node membership
- Data redistribution and key transfer
- Systems-level networking implementation

---

### [Distributed Message Passing](https://github.com/emmanuel-sp/Distributed-Message-Passing)
**Fault-tolerant publish-subscribe messaging system in C++**

Built a coordinator-based messaging system with reconnect handling and TTL-based message expiration.

**Highlights**
- Publish-subscribe architecture
- Socket-based communication
- Fault-tolerant reconnect support
- Time-sensitive message delivery semantics
